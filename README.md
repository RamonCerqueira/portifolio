# Ramon Cerqueira - Next.js + Tailwind (Projeto Gerado)

Conteúdo:
- Next.js (app router) skeleton
- TailwindCSS configuration
- Framer Motion animations (navbar, hero, cards)
- Page transitions, scroll progress bar, theme (dark/light) toggle
- API route example: `/app/api/contact/route.ts` (handler that logs payload)
- Placeholder images in `public/images`
- ZIP gerado em `/mnt/data/Ramon Cerqueira-site.zip`

Como usar (local):
1. Instale dependências: `npm install`
2. Rode em desenvolvimento: `npm run dev`
3. Abra: http://localhost:3000

Observações:
- Substitua imagens em `public/images`.
- Integre o `app/api/contact/route.ts` com sua solução de envio de e-mail (SendGrid, Mailgun) ou backend.
